# Maintenance and Inspection 

Regular inspection and maintenance are crucial to ensure a robust and durable system over time. These activities help extend the system's operational life and reduce repair costs.


## <p style={{ color: 'purple' }}>1. Inspection Steps</p>

- *Step 1:* Check the Connections of the Devices in the Electrical Cabinet

You can view the electrical cabinet connection diagram [here](https://drive.google.com/file/d/1P_Ra2Gr5d-fQH4x_3uBTKaf51-Me_TBY/view?usp=sharing).

<div style={{  borderLeft: '4px solid yellow',  backgroundColor: '#FFFFE0',  padding: '10px',  borderRadius: '5px'}}>
  <strong style={{ color: 'Red' }}>⚠️ CAUTION</strong>
  <p style={{ color: 'blue' }}>Before proceeding with the inspection, ensure that all `electrical safety criteria` are met.</p>
</div>

- *Step 2:* Verify the Operation of the Modbus RTU 32CH Relay

> This is a critical component of the cabinet, and we have created a tool to facilitate this check. You can download the tool [here](linktool).

<div style={{  borderLeft: '4px solid yellow',  backgroundColor: '#FFFFE0',  padding: '10px',  borderRadius: '5px'}}>
  <strong style={{ color: 'Red' }}>⚠️ CAUTION</strong>
  <p style={{ color: 'blue' }}>Only proceed with this step after completing Step 1 and ensuring the electrical system is `entirely safe`.</p>
</div>

### <p style={{ color: 'purple' }}>Tool Usage Instructions</p>
- ***Step 1:** Connect the Main Control Circuit to Your PC* :computer:
    - Connect the main control circuit of the cabinet to your personal computer using the appropriate cables and interfaces.
- ***Step 2:** Launch the Tool*
    - Click on the CaCoiot icon to open the tool.
    - If the tool starts successfully, it indicates a successful connection. If the tool does not appear on the screen, check all connections and try again.
- ***Step 3:** Using the Tool*
    - Once the tool interface is displayed, it will look similar to the provided image. Here, you can monitor and control the on/off states of the 32 channels of the Modbus RTU Relay 32CH.
    - Every time you toggle a relay on or off, the action will be logged in the history column within the tool.

<div style={{ borderLeft: '4px solid green',  backgroundColor: '#f0fff4',  padding: '10px',  borderRadius: '5px' }}>
   **Identifying Faults:**  
   - When you toggle any relay, if the relay functions correctly, you should see an indicator light at the relay's location and hear a distinct "click" sound.
   - If the light does not illuminate or there is no sound, it may indicate a malfunction with that specific relay.

</div>
